## Project Summary
A rebranded version of the Ingressos Sul & Serra website, now called Ingressos Sul & Serra. It is a Brazilian e-commerce marketplace for travel, leisure, and gastronomy deals in the South and Mountain regions. The project focuses on a high-conversion, grid-heavy layout with an authentic golden branding and clear price signaling.

## Tech Stack
- Framework: Next.js (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- Components: Lucide React (Icons), Next.js Image
- Fonts: Montserrat

## Architecture
- `src/app/`: Contains the main page and global styles.
- `src/components/sections/`: Contains all the cloned sections of the website (Header, Footer, Hero, Carousels, etc.).
- `src/components/ui/`: Contains UI-specific components like the Cookie Notice.
